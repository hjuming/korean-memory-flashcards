# 下一階段開發建議

## 目前版本狀態

目前版本已具備本機可用的 Flashcard Studio：

- 首頁保留圖像聯想閃卡練習。
- 側邊欄管理新增卡片、記憶庫與學習進度。
- 輸入韓文後可生成聯想密碼、圖像畫面與圖像 Prompt。
- 可上傳圖片、匯入匯出 JSON。
- 可點選「發音」播放韓文朗讀。
- 已預留 Cloudflare Pages Function `/api/generate-card` 作為 AI API 接入點。

## 下一階段優先順序

### 1. 接入 AI 生成

目標：讓使用者輸入韓文後，取得更準確的中文意思、發音拆解、聯想密碼與圖像 Prompt。

建議做法：

- 在 Cloudflare Pages 設定 `OPENAI_API_KEY`。
- 視成本與品質設定 `OPENAI_MODEL`。
- 為 `/api/generate-card` 加入輸入長度限制、錯誤訊息與簡易 rate limit。
- 將 AI 回傳欄位固定為 JSON schema，降低格式錯誤。

驗收標準：

- 沒有 key 時仍可使用 fallback。
- 有 key 時可回傳結構化草稿。
- 前端永遠不出現 API key。

### 2. 升級發音品質

目前使用瀏覽器內建 Web Speech API，優點是免費且不需後端；限制是音色依裝置而定。

建議做法：

- 第一階段保留內建發音。
- 第二階段可新增「高品質發音」模式，透過 Cloudflare Function 呼叫第三方 TTS。
- TTS 音檔可暫存到 Cloudflare R2，避免重複產生費用。

驗收標準：

- 每張卡可有 `audioUrl`。
- 無音檔時使用內建發音 fallback。
- 音檔與 API key 不進入前端原始碼。

### 3. 圖片與資料儲存升級

目前圖片以 data URL 存在 `localStorage`，適合小量自用；大量圖片會佔空間。

建議做法：

- 圖片改存 Cloudflare R2。
- 卡片資料可升級到 Supabase。
- 保留 JSON 匯出作為手動備份。

驗收標準：

- 本機資料可匯入雲端。
- 圖片不再塞爆 localStorage。
- 未登入或離線仍保留基本練習能力。

### 4. 學習排程強化

目前已有簡易待複習與連續記住。

建議做法：

- 將排程改為 spaced repetition：新卡、學習中、熟悉、長期記憶。
- 增加每日練習清單。
- 增加「今天完成」與「連續天數」。

驗收標準：

- 使用者打開首頁就知道今天要練哪些卡。
- 每次標記都會更新下一次複習時間。
- 進度面板可看見一週內待複習數量。

### 5. PWA 與行動體驗

目前已具備 icon 與 manifest，可再補強離線使用。

建議做法：

- 加入 service worker 快取靜態資源。
- 增加安裝提示與離線狀態提示。
- 手機版優化卡片高度與操作按鈕密度。

驗收標準：

- 網路不穩時仍可練習既有卡片。
- 更新後能正常刷新資源版本。
